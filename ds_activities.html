Other Software

The Nintendo DS Lite offers a good bunch of software features that favor multiple players. Some of these include:

  • Pictochat, which allows multiple users to draw and send messages to one another
  • DS game sharing, which allows users with an actual copy of a game to allow users without the game to play it anyways via the internet
